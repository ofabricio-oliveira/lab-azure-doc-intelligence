📄 Extrator de Documentos

Azure Document Intelligence LAB

Envie um documento (PDF, PNG, JPG) e escolha o modelo prebuilt do Azure AI Document Intelligence para analisá-lo.

O que cada modelo faz? Estes são modelos prebuilt do Azure Document Intelligence:
  • 🔍 OCR Read (prebuilt-read) — Extração de texto puro via OCR. Reconhece texto impresso e manuscrito, detecta idiomas e retorna páginas, linhas e palavras. É a base dos demais modelos.
  • 🧾 Recibo (prebuilt-receipt) — Extrai campos estruturados de comprovantes de venda: comerciante, data, itens, total, impostos, forma de pagamento.
  • 📃 Fatura (prebuilt-invoice) — Extrai campos estruturados de documentos de cobrança: fornecedor, cliente, itens, total, vencimento, condições de pagamento.
